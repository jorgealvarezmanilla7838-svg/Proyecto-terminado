 Descripción
NefroCare es una WebApp Full-Stack diseñada para optimizar la atención en el consultorio de nefrología. Permite a los pacientes agendar citas, recibir educación renal mediante un chatbot interactivo y a la administración gestionar la agenda médica de manera eficiente.
